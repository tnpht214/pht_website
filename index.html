📚 Học Tool

🌐 HTML

HTML (HyperText Markup Language - Ngôn ngữ đánh dấu siêu văn bản) đóng vai trò là bộ khung xương cơ bản cho mọi trang web trên internet. Mã HTML dùng để cấu trúc và sắp xếp các thành phần nội dung trên trình duyệ

🎨 CSS

CSS (Cascading Style Sheets) dùng để trang trí và làm đẹp cho trang web. Nếu HTML là bộ khung xương, thì CSS chính là quần áo, đầu tóc và lớp trang điểm giúp trang web trở nên đẹp mắt, dễ nhìn hơn..

⚡ JavaScript

Java là một ngôn ngữ lập trình giúp tạo ra nhiều phần mềm và ứng dụng.Dưới đây là những việc chính mà Java giúp ích:Làm ứng dụng điện thoại: Dùng để viết các ứng dụng trên hệ điều hành Android.Làm trang web và hệ thống lớn: Giúp các ngân hàng, ứng dụng mua sắm chạy mượt mà và bảo mật.Chạy trên mọi máy tính: Code Java chạy được trên cả Windows, Mac và Linux.Dễ tìm việc làm: Người giỏi Java luôn được các công ty săn đón với lương cao

💻 VS Code

VS Code là phần mềm lập trình được sử dụng phổ biến nhất hiện nay.
